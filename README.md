# BookKaro - A Book Barter System Application

**BookKaro** is an Android application designed to streamline the buying, selling, donating, and exchanging of books. It features account management, book browsing, and category-based selections, creating a digital space where users can find books they need while sharing those they no longer use.

## Features

* **Buy, Sell, & Donate:** A comprehensive platform to trade or donate educational and fictional books.
* **User Accounts:** Secure account creation and login functionality to manage listings.
* **Category-wise Browsing:** Easily find books by category (Engineering, Medical, Fiction, etc.).
* **Book Details:** View detailed information about book condition, price, and seller notes.
* **Contact Support:** Integrated support option for user queries.
* **Intuitive UI:** Custom ListView implementation for a smooth user experience.

## Tech Stack

* **Language:** Java / XML
* **Platform:** Android
* **Build Tool:** Gradle
* **Database:** SQLite (Local storage for user and book data)
* **IDE:** Android Studio

## How to Run

To run this project on your local machine, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/shruteesalpe/BookKaro---A-Book-Barter-System-App.git
   ```

2. **Open in Android Studio:**
   * Launch Android Studio.
   * Select **Open** and navigate to the cloned folder.

3. **Sync Project:**
   * Wait for Gradle to sync the project and download necessary dependencies.

4. **Run the App:**
   * Connect an Android device via USB or use an Android Emulator (AVD).
   * Click the green **Run** button (Play icon) in the toolbar.
